- Install dependencies set flow
  - Client -> RequestHandler: take this nix file
  - RequestHandler -> Database: store this dependencies set
    - Dependencies set: {
        id: string
        name: string
        description: string
        nix_file_content: string
        status: 'INSTALLING | INSTALLED' // for any worker
      }
    - Dependencies installations {
        id: string
        dependencies_set_id: string
      }
  - RequestHandler -> InMemoryStore: publish this dependencies set installation message
  - InMemoryStore -> Worker: take this dependencies set installation message
  - Worker -> Worker: build the nix file (affects WorkerCache)
  - Worker -> WorkerCache: store the translated compile_setup, run_setup
  - Worker -> Database: sign the dependencies set object with my id

- Before the worker starts processing a submission, check if you have signed its dependencies

- Get dependencies set status flow
  - Client -> RequestHandler: dependencies set id
  - RequestHandler -> Database: get dependencies set with id
  - RequestHandler -> Client: dependencies set

- Get all dependencies set
  - Client -> RequestHandler: where ma dependencies
  - RequestHandler -> Database: where ma dependencies
  - RequestHandler -> Client: all dependencies
